Análisis Exploratorio - House Prices

Introducción

Este repositorio está hecho con el fin de entrenar la exploración de datos, utilizando como guía los cuadernos de Kaggle Comprehensive data exploration with Python y House Prices - Advanced Regression Techniques. Además, he traducido el contenido para practicar mi inglés.

Tarea

Nuestra tarea es predecir el precio de venta de cada casa. Para cada Id en el conjunto de prueba, debes predecir el valor de la variable SalePrice.

Instalación

Para realizar el análisis, es necesario instalar la API de Kaggle. Si no la tienes instalada, ejecuta:

pip install kaggle

Después, descarga el archivo kaggle.json desde la configuración de tu cuenta de Kaggle y colócalo en la carpeta .kaggle:

move ~/downloads/kaggle.json ~/.kaggle/

Descarga el conjunto de datos con:

kaggle competitions download -c house-prices-advanced-regression-techniques

Mueve y extrae el archivo ZIP:

move ~/downloads/house-prices-advanced-regression-techniques.zip ~/onedrive/Data/
Expand-Archive -Path ~/onedrive/Data/house-prices-advanced-regression-techniques.zip -DestinationPath ~/onedrive/Data/

Resultados

Al extraer los archivos, obtendremos:

├── Data/
│   ├── train.csv
│   ├── test.csv
│   ├── sample-submission.csv
│   └── data_description.txt
